# Ecommerce Sales Dashboard

This repository contains an interactive **Ecommerce Sales Dashboard** built to analyze and visualize sales performance across multiple dimensions such as state, category, customer, payment mode, and time period.

---

##  Features

- **Summary Metrics**
  - Total Sales Amount: 161K
  - Total Profit: 26K
  - Total Quantity Sold: 2008
  - Average Order Value (AOV): 44K

- **Filters**
  - Quarter selection (Q1, Q2, Q3, Q4)
  - State selection dropdown

- **Visualizations**
  -  Sales by State (Maharashtra, Madhya Pradesh, Delhi, Uttar Pradesh)
  -  Quantity by Category (Furniture, Electronics, Clothing)
  -  Profit by Month (Jan–Mar)
  -  Sales by Customer (Harivansh, Shiva, Madhav, Santa)
  -  Quantity by Payment Mode (Credit Card, EMI, Debit Card, UPI, COD)
  -  Profit by Sub-Category (Printers, Phones, Accessories, Bookcases, Saree)

## Project Structure

- `data/` → Raw and cleaned datasets
- `dashboard/` → Dashboard files (Power BI)
- `README.md` → Documentation

## Tech Stack

- **Data Analysis:** Python 
- **Visualization:** Power BI 
- **Version Control:** Git & GitHub
